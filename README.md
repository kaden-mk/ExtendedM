# ExtendedM
ExtendedM is an extended framework for FiveM which reaches to contain all base GTA Online features, whilst also improving upon some things that generally make user experience better.

ExtendedM comes with its own data-saving system although it doesn't use sql databases, rather json files since this was made for small servers.

## Features
* Built-in native-like interface system
* Store system
* Easy to access API
* ATMS & Economy
